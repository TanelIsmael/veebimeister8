== Description ==
Bizplan is an enticing and visually stimulating, tender and polished, professional and responsive WordPress creative multipurpose theme based on Bootstrap framework. Bizplan is modern theme fit for creative individuals, agency, blog, business, company, corporate, and professional portfolio. The design is very elegant and modern, and also very easy to customize. Build your own website with our awesome Bizplan! Demo URL: https://keonthemes.com/theme-demo/?id=NjM5fGJpenBsYW58Qml6cGxhbg= and for more detail: https://keonthemes.com/downloads/bizplan

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Bizplan includes support for WooCommerce, Contact From 7 and Jetpack.
